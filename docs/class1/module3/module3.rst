Module 3: High Availability
===========================

**Goal:**

In this module, you will learn how to complete High Availability setup for BIG-IQ CM. 
Starting in BIG-IQ v7.0, you have the option to chose **Manual** or **Auto** failover. 

The steps are identical for AWS and VMware.
